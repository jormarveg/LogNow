# Referencias

## APIs y servicios externos

- [IGDB API](https://api-docs.igdb.com/)  
  Documentación utilizada para consultar videojuegos, compañías, géneros, plataformas, fechas de lanzamiento, portadas y puntuaciones externas.

- [Twitch Developers - Authentication](https://dev.twitch.tv/docs/authentication/)  
  Documentación consultada para obtener el token de acceso necesario para realizar peticiones a la API de IGDB.

## Backend y base de datos

- [PHP](https://www.php.net/)  
  Lenguaje utilizado para desarrollar la lógica de servidor de la aplicación.

- [PDO](https://www.php.net/manual/es/book.pdo.php)  
  Extensión de PHP utilizada para realizar consultas preparadas y acceder a la base de datos de forma segura.

- [MariaDB](https://mariadb.org/)  
  Sistema gestor de base de datos relacional utilizado para almacenar usuarios, videojuegos, reseñas, listas, géneros, plataformas y demás información de la aplicación.

- [Nginx](https://nginx.org/)  
  Servidor web utilizado en el entorno Docker para servir la aplicación y redirigir las peticiones PHP al contenedor correspondiente.

- [FPDF](http://www.fpdf.org/)  
  Librería utilizada para generar documentos PDF desde PHP.

## Frontend

- [MDN Web Docs](https://developer.mozilla.org/es/)  
  Referencia general utilizada para consultar documentación sobre HTML, CSS, JavaScript, eventos, formularios y APIs del navegador.

- [MDN - CSS Grid](https://developer.mozilla.org/es/docs/Web/CSS/CSS_grid_layout)  
  Documentación consultada para la creación de layouts en rejilla, como catálogos, perfiles y listados.

- [MDN - Flexbox](https://developer.mozilla.org/es/docs/Web/CSS/CSS_flexible_box_layout)  
  Documentación consultada para la alineación y distribución de elementos en menús, cabeceras, botones y tarjetas.

- [MDN - Media queries](https://developer.mozilla.org/es/docs/Web/CSS/CSS_media_queries)  
  Referencia utilizada para implementar el diseño responsive de la aplicación.

- [MDN - Fetch API](https://developer.mozilla.org/es/docs/Web/API/Fetch_API)  
  Documentación consultada para realizar peticiones asíncronas desde JavaScript.

- [jQuery](https://jquery.com/)  
  Librería utilizada para simplificar algunas operaciones de manipulación del DOM, eventos y peticiones AJAX.

- [Font Awesome](https://fontawesome.com/)  
  Librería de iconos utilizada en la interfaz, especialmente en menús, botones, estados y sistema de puntuación con estrellas.

- [Google Fonts](https://fonts.google.com/)  
  Servicio utilizado para seleccionar e incorporar las tipografías de la aplicación.

- [Fancyapps Carousel](https://fancyapps.com/)  
  Librería utilizada para los carruseles de juegos y contenidos destacados.

## Despliegue, documentación y control de versiones

- [Docker](https://www.docker.com/)  
  Herramienta utilizada para preparar un entorno de ejecución reproducible mediante contenedores.

- [AWS EC2](https://aws.amazon.com/ec2/)  
  Servicio contemplado para el despliegue de la aplicación en una instancia de servidor.

- [Git](https://git-scm.com/)  
  Sistema de control de versiones utilizado durante el desarrollo del proyecto.

- [GitHub](https://github.com/)  
  Plataforma utilizada para alojar el repositorio del proyecto.

- [GitHub Pages](https://pages.github.com/)  
  Servicio utilizado para publicar la documentación del proyecto.

- [Just the Docs](https://just-the-docs.com/)  
  Tema utilizado para generar la documentación en GitHub Pages a partir de archivos Markdown.
